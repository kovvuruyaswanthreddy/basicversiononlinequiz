📑 Faculty Comprehensive Guide

Everything you need to know about managing exams, uploading spreadsheets, and tracking student performance on the MITS Assessment Portal.

🏫 Academic Exams Module

1. Initial Setup: First, navigate to Academic Exams > Create Quiz. You must create a unique Quiz ID (e.g., 23-CSE-JAVA-01) before you can upload questions.
2. Question Management: Use the Upload Questions tool. The system will map your Excel rows to database entries. Once questions are uploaded for a specific Quiz ID, they cannot be modified to maintain integrity.
3. Quiz Activation: Go to Activate Quiz. Here you bridge the quiz to a specific batch. Select the Department, Year (1-4), and Section (A-J). Set a timer in minutes.
4. Result Publishing: Students cannot see their scores even after submission until you Publish Results. This prevents the sharing of answers during active exam windows.

🏆 Events & Hackathons

1. Create Event: Events are independent of academic batches. Create an Event ID (e.g., CODERALLY-2025).
2. Participant Registration: Unlike academic students, event students are uploaded specifically to an event. Use the Upload Event Students feature.
3. Event Quiz Flow: Activate quizzes within the event. These quizzes are only accessible to students whose Roll Numbers are registered for that specific Event ID.

📑 Result Management

View Class Results: Filter by Quiz ID, Dept, Year, and Section. The table shows Rank, Gained Marks, Total Marks, and Submission Time.
Excel Export: Click the "Download Excel" button in the results view to get a formatted spreadsheet of the student performances including their pass/fail status.

🛡️ Security & Anti-Cheat

Focus Tracking: If a student attempts to leave the tab or minimize the browser, a warning is issued. On the 3rd attempt, the exam is INSTANTLY submitted and locked.
Idempotency: The portal prevents double-submissions. If a network failure occurs, students can refresh and their previous answers will be auto-recovered from the local session.
