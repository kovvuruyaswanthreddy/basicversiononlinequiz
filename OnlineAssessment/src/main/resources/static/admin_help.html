🛠️ Enterprise Administration

Super Admin Privilege: You are accessing the core configuration layer. Actions taken here immediately affect the entire database and user access levels.

1. Institutional Oversight

College Lifecycle: Manage institutional access. You can register new colleges under the MITS umbrella, update their branding, and monitor their active user counts.
Global Kill-Switch: If an institution violates policy or subscription terms, setting their status to DEACTIVATED will lock all related Faculty, Student, and Admin accounts instantly.

2. Faculty Governance

Maintain the backbone of the assessment engine by managing faculty credentials.

3. Global Schematics

Departmental Hierarchy: Create new departments (e.g., AI&DS, CSBS) to update the dropdown selections across the entire portal. Ensure spellings match your Excel data exactly.
Privilege Matrix: Grant or revoke high-level features like Real-time Proctoring, Advanced Analytics, or Event-Only Portals for individual colleges.

4. Technical Audits

Data Synchronization: If data appears mismatched, perform an "Audit Sync" via the dashboard to recalibrate relational mappings between students and their respective departments.
